# Installation

## Requirements

- Python 3.9+
- PyTorch 1.10+

## Install via pip

```bash
pip install tensight
```

## Install from source

```bash
git clone https://github.com/JonathanDray/tensight.git
cd tensight
pip install -e .
```

## Verify installation

```python
import tensight
print(tensight.__version__)
```
